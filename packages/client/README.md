# client

React frontend running on Vite, served on port `5173`. API requests to `/api` are proxied to the server on port `3000`.

## Development

```bash
bun run dev
```

## Scripts

- `dev` — start the dev server with HMR
- `build` — type-check and build for production
- `preview` — preview the production build
- `lint` — run ESLint
